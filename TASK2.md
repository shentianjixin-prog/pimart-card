# CNBOX STORE 改进任务（参考同类卡牌网站）

参考网站：Cardshop Serra（日本最大卡牌专卖）、CARDOTAKU（日本卡牌海外销售）、Hareruya2（宝可梦专门店）

---

## 一、商品卡片升级（ProductCard.tsx）

### 1. 加"快捷加购"按钮
鼠标悬停时卡片底部滑出"+ 加入购物车"按钮，不用进详情页。
- 只在 `stock > 0` 且非预售时显示
- 使用 `group-hover:translate-y-0` 动画

### 2. 库存紧张提示
在价格下方显示库存状态：
- `stock <= 3` → 显示橙色文字"仅剩 X 件"
- `stock === 0` → 已有 SOLD OUT 蒙层，保持不变

### 3. NEW 标签
`createdAt` 在 7 天内的商品，左上角显示绿色 `NEW` 角标（和预售的橙色 `预售` 角标位置错开，分左右）。

---

## 二、首页升级（page.tsx + 新组件）

### 4. 首页轮播 Banner（替换现有 HomeHero）
在 `HomeHero.tsx` 里做一个自动轮播 Banner（3 张静态幻灯片，每 4 秒切换，带左右箭头和底部圆点）：
- 幻灯片1：宝可梦原盒主题（蓝紫渐变背景）
- 幻灯片2：火影忍者主题（橙红渐变背景）
- 幻灯片3：预售说明（强调 5-7 天发货）

> 纯 CSS + React state 实现，不引入新依赖。

### 5. "本周热销"区块
在推荐商品区下方新增"本周热销"区，展示 `stock` 最少（销售最旺）的 4 个有库存商品，带"热销"红色角标。

---

## 三、商品详情页升级（products/[slug]/page.tsx）

### 6. 多图轮播
`images` 字段是逗号分隔的多图 URL，现在只显示第一张。改为：
- 左侧大图展示当前选中图片
- 下方缩略图行（最多 5 张），点击切换大图
- 用 `useState` 管理当前选中索引，不引入新依赖

### 7. 发货流程时间线
在"发货说明"模块下方加一个横向时间线图示：

```
[下单支付] → [备货打包] → [发货出库] → [收货]
  当天        1-3天        3-5天        5-7天
```

用纯 Tailwind 实现，4 个节点加连接线。

---

## 四、新增"公告/动态"区块（首页底部，新建组件 Announcements.tsx）

参考 Hareruya2 的博客区、CARDOTAKU 的新品资讯区。

在首页商品列表上方（筛选栏外面）加一个静态公告栏，显示 3 条公告：
```
📢 【新品预告】简中151 BOX 即将开启预售，敬请期待！
📦 【发货通知】6/20 前下单的订单已于 6/25 全部发出
✅ 【正品保证】本店所有商品均为正规渠道进货，假一赔十
```
用折叠式设计（默认展开，可点击关闭），存在 `localStorage` 里记住是否关闭过。

## 五、信任度模块（新建组件 TrustBadges.tsx）

参考 CARDOTAKU 的"精心包装承诺"、Serra 的"真伪鉴定"。

在首页 Hero 下方加一行 4 个信任徽章（横排，移动端 2x2）：

| 图标 | 标题 | 副文字 |
|------|------|--------|
| 🛡️ | 正品保证 | 全渠道正规进货 |
| 📦 | 专业包装 | 气泡膜+硬壳保护 |
| ⚡ | 快速发货 | 5-7 个工作日 |
| 💬 | 售后保障 | 收货 7 天内可处理问题 |

## 六、邮件订阅栏（Footer 上方，新建组件 NewsletterBar.tsx）

参考 CARDOTAKU 的邮件订阅入口。

在 Footer 上方加一条订阅栏：
```
🔔 订阅新品通知  [邮箱输入框]  [订阅]
   第一时间获取新品预售信息
```
点击"订阅"后显示"✅ 已订阅，感谢！"（前端静态提示即可，不需要后端实现）。

---

## 七、管理后台：库存快速编辑（admin/products/page.tsx）

参考实际运营需求。

在管理后台商品列表的每一行，加一个"库存"内联编辑：
- 显示当前库存数字，点击变成输入框
- 失焦后直接调用 Server Action 保存，不用进入编辑页
- 保存成功闪绿色，失败闪红色

## 八、执行顺序

1. `TrustBadges.tsx` + 在首页 Hero 下引入（最简单，先做）
2. `ProductCard.tsx` 升级（快捷加购 + 库存提示 + NEW 标签）
3. `HomeHero.tsx` 改为轮播 Banner
4. 商品详情页多图轮播
5. 发货时间线
6. "本周热销"区块
7. `Announcements.tsx` 公告栏
8. `NewsletterBar.tsx` 邮件订阅
9. 后台库存快速编辑

## 注意事项

- 不引入任何新 npm 依赖（用 React state + Tailwind 实现所有交互）
- 所有新组件放在 `src/components/` 下
- 移动端优先，确保手机上正常显示
- 颜色风格沿用现有配色：cyan-400 / violet-500 / orange-500

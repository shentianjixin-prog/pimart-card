# 航海王简中官方商品图

本目录只保存航海王卡牌对战简中官网公开的商品素材，用于统一商品目录的主图状态。

- EBC-02～04、OPC-01～16、PRBC-01～02：官网商品目录中的官方补充包正面图。
- `ebc-01-pack.png`：官网“特别补充包 纪念收藏合集”正面图；官网未标注 EBC 编号，文件名只作本地排序，不在商品标题中冒充官方编号。
- STC-01～04：从官网官方牌组合图中按原像素裁切，不进行生成式重绘。
- 其余 STC / 基本卡组：官网商品目录中的官方牌组图。官网未公开编号的“变档5档”“艾斯和纽哥特”“路飞＆艾斯”不强行写入推测编号。

来源：

- 官网：https://www.onepiece-cardgame.cn/
- 官网商品目录 API：https://onepieceserve.windoent.com/products/productsinfo/webList
- 官方素材 CDN：https://source.windoent.com/OnePiecePc/Picture/
- STC-01～04 原始合图：`1667530531560基本卡组轮播图（更新为发售中）.jpg`

目录页约定：EBC、OPC 补充包系列统一展示补充包正面图，STC 系列统一展示牌组盒图；商品实际销售规格仍由页面上的 `boxType`（原盒、散包、原箱等）明确标示。
